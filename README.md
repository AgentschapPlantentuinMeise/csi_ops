# CSI Operations - Standard Operating Procedures

This repository contains the open Standard Operating Procedures (SOPs)
for the **Crop Source Investigations (CSI)** laboratory
at Meise Botanic Garden.  
All procedures are maintained in Markdown format to ensure
transparency,
reproducibility, and long-term accessibility.

## Structure

- `sop/` — All SOPs, organised by process or instrument.
- `assets/` — Shared diagrams, images, and templates.
- `sop-index.md` — Human‑readable overview of all SOPs.
- `CONTRIBUTING.md` — Guidelines for proposing changes.

## Goals

- Provide clear, reproducible workflows for laboratory and analytical processes.
- Enable open collaboration with partners and the scientific community.
- Maintain versioned, auditable documentation using Git and GitHub.

## Using the SOPs

You can browse the SOPs directly in GitHub, or clone the repository:

    git clone https://github.com/AgentschapPlantentuinMeise/csi_ops.git


If you wish to build a documentation website (optional), see the instructions in `CONTRIBUTING.md`.
